# Python_Diwali_Sales_Analysis
This project analyzes Diwali sales data to understand customer purchase behavior, top-selling product categories, gender/age spending patterns, and overall sales trends. Using Python libraries such as Pandas, NumPy, Matplotlib, and Seaborn, the project delivers clear insights with visualizations.

Female customers contribute the highest total purchases, indicating strong buying interest during festive sales.

Age groups 26–35 and 36–45 show the highest spending, useful for targeted marketing.

Product categories like Clothing, Electronics, and Accessories emerged as top sellers.

Majority of the customers are married working professionals, suggesting stable purchasing power.

Certain states and cities show significantly higher sales, highlighting strong regional markets.

This analysis provides actionable insights to improve sales strategy, customer engagement, and inventory planning for festive seasons like Diwali. The findings help businesses make data-driven decisions that maximize revenue and enhance customer satisfaction.
